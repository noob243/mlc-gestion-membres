🚀 DEMARRAGE PRODUCTION EN 3 ETAPES:


ETAPE 1: Créer Supabase (5 minutes)
──────────────────────────────────
1. Allez sur: https://app.supabase.com
2. Créez un nouveau projet
3. Exécutez le SQL fourni dans le guide
4. Récupérez vos API Keys


ETAPE 2: Pousser vers GitHub (2 minutes)
────────────────────────────────────────
1. Créez un repo GitHub
2. Utilisez les commandes:
   
   git add .
   git commit -m "Prepare for Netlify deployment"
   git push origin main


ETAPE 3: Configurer Netlify (5 minutes)
────────────────────────────────────────
1. Allez sur: https://app.netlify.com
2. Connectez GitHub
3. Sélectionnez votre repo
4. Ajoutez variables d'environnement:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
5. Validez → Déploiement automatique!

📞 TROUBLESHOOTING:


❌ ".env not found"
   → Créez .env localement avec:
     VITE_SUPABASE_URL=...
     VITE_SUPABASE_ANON_KEY=...


❌ "Netlify Functions return 500"
   → Vérifiez logs: Netlify → Functions → members
   → Vérifiez que Supabase URL/Key correctes


❌ "Database permission denied"
   → Supabase → Table Editor → Check RLS policies
   → Assurez-vous les politiques permettent INSERT


❌ "Frontend builds mais backend API fail"
   → Utilisez .netlify/functions URLs automatiquement
   → Vérifiez CORS headers dans netlify.toml
