🚀 DEMARRAGE EN 30 SECONDES:

   1. Double-cliquez sur: lancer.bat
      (ou lancer_debug.bat si vous voulez voir tous les logs)

   2. Attendez 5-10 secondes (Docker démarre, npm compile...)

   3. Le navigateur s'ouvrira automatiquement sur:
      → http://localhost:3000

   4. C'est tout ! L'application est lancée.


📍 URLS D'ACCES:

   Frontend:   http://localhost:3000
   Backend:    http://localhost:5000/api/members
   Database:   postgresql://postgres:postgres@localhost:5434/mlc_db


🔧 EN CAS DE PROBLÈME:

   ❌ Frontend inaccessible?
      → Utilisez lancer_debug.bat pour voir les logs
      → Vérifiez que le port 3000 est libre
      → Relancez: taskkill /F /IM node.exe && lancer_debug.bat

   ❌ Erreur base de données?
      → Vérifiez Docker: docker compose ps
      → Relancez Docker: docker compose restart postgres
      → Réinitialisez: npm run init-db

   ❌ npm install échoue?
      → Utilisez: npm install --legacy-peer-deps
      → (C'est normal, qrcode.react nécessite ce flag)
